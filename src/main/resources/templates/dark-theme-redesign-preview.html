Dark theme, take two

The current dark mode leans on a navy-tinted background and borders, so everything reads as "blue on blue." This version keeps the same brand navy for the sidebar and hero bands, but swaps the page background, cards, and borders to a neutral charcoal — so the teal/amber/green accents actually pop instead of blending in.
